# PaperMind — 智能学术论文研究平台

## **开发人：gxy | Git 提交次数：11 次**

## 平台简介

本项目 PaperMind 是一个面向研究生和科研人员的 AI 论文管理与分析系统。它将 PDF 论文转化为可检索、可对话、可可视化的知识库，帮助研究者高效阅读、理解和发现学术文献。

## 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端 | Next.js 16 + React 19 + TailwindCSS |
| 后端 | FastAPI + LangChain |
| 向量数据库 | ChromaDB |
| 知识图谱 | Neo4j |
| 大语言模型 | OpenAI 兼容接口（DeepSeek / GPT-4o） |
| 会话存储 | SQLite |

## 核心功能

### 1. 论文管理
- 上传 PDF，自动解析全文并提取元数据（标题、作者、摘要）
- 支持两种解析引擎：PyPDF（快速）/ MinerU（高精度，支持双栏和公式）
- 在线 PDF 阅读器

### 2. 智能问答（RAG）
- 基于论文内容的多轮对话，支持流式输出
- 混合检索：向量语义搜索 + BM25 关键词搜索 + RRF 融合 + LLM 重排序
- 查询改写（解析代词/上下文指代）
- 对话记忆管理（自动摘要 + Token 预算控制）
- 会话持久化，支持历史记录查看与管理

### 3. 知识图谱
- LLM 自动抽取论文中的方法、问题、概念、数据集及其关系
- 交互式力导向图可视化（React Flow + D3）
- 支持单篇论文子图 / 多篇论文对比视图
- 关键词共现网络

### 4. 论文推荐
- 基于已有论文的研究画像，从 Semantic Scholar 检索近期相关文献
- 支持按时间范围筛选（近1年 / 半年 / 3个月）
- CCF 等级标注与过滤（A / B / C 类会议期刊）

### 5. 研究统计
- 论文数量、方法数、主题数、关系数总览
- 研究主题频率分布图
- 论文覆盖度热力图

## 数据流

```
PDF 上传 → 全文解析 → 元数据提取 → 文本分块(800字)
    ↓                                     ↓
知识图谱抽取 → Neo4j              向量嵌入 → ChromaDB + BM25 索引
    ↓                                     ↓
图谱可视化 / 统计分析            混合检索 → 多轮对话 → 流式回答
```

## 页面结构

| 页面 | 功能 |
|------|------|
| Library | 论文库管理，上传/删除/在线阅读 |
| Chat | 基于论文的多轮智能问答 |
| Knowledge | 知识图谱交互式可视化 |
| Discover | AI 论文推荐，按等级和时间筛选 |
| Stats | 研究数据统计面板 |

## 快速启动

```bash
# 后端
pip install -r requirements.txt
uvicorn server:app --host 0.0.0.0 --port 8000

# 前端
cd frontend
npm install
npm run dev
```

需要配置 `.env` 中的 OpenAI API Key 和 Neo4j 连接信息。
